# 🏦 Debit & Credit Analytics Dashboard

A comprehensive **Bank Transaction Dashboard** built to analyze **Debit and Credit activity**, financial trends, and customer behavior using multiple tools.

> 🚀 Real-world financial analytics dashboard with KPI tracking and interactive filtering.

---

## 📌 Project Overview

This project focuses on analyzing **bank transactions** to uncover insights related to:

- Total credited vs debited amount  
- Customer spending behavior  
- Branch-wise transactions  
- Monthly financial trends  

The dashboard provides a **clear financial overview** for better decision-making.

## ✨ Dashboard Features

### 📊 Transaction Analysis
- Branch-wise transaction comparison  
- Transaction type distribution (Debit vs Credit)  
- Transaction purpose analysis  

### 👥 Customer Insights
- Top 5 customers by total transaction amount  
- Credit vs Debit comparison per customer  

### 📈 Trend Analysis
- Monthly transaction trends for 2024  
- Time-based financial performance tracking  

### 🎛️ Interactive Filters
- Bank selection  
- Month-wise filtering  
- Dynamic dashboard updates
